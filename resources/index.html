Recommended Resources

Brush up on skills or learn at your own pace with the resources below.

Top Resources

The first resources we recommend to interested students.

  • Dash (HTML & CSS):
    A great, interactive introduction to HTML and CSS that we recommend for students who are new to programming. Helps get something working on your screen in minutes
  • /r/learnprogramming (General):
    A subreddit for learning new things. The right sidebar and FAQ are chock-full of links to resources, books, and courses with awesome stuff to learn. A great starting point for looking up new things!
  • Codecademy (HTML & CSS, JavaScript, Python, Ruby, etc.):
    Free, interactive online courses in popular languages. An excellent starting point for learning
  • Project Euler (language-agnostic math problems with programming solutions):
    A site with mathematics problems requiring algorithmic solutions. An excellent way to challenge yourself, or to get you up to speed with a new programming language

Assignments/Challenges

  • Mega Project List — a list of programming projects to work on, for programmers of all levels
  • Advent of Code — a series of small, fun, self-contained programming puzzles for a variety of skill levels
  • Project Euler — math problems requiring algorithmic solutions; a great way to get up to speed with a new programming language!
  • Nifty Assignments — a collection of nifty programming assignments collected by Stanford

HTML & CSS

Learning HTML and CSS is the first step towards creating websites.

  • Dash — highly recommended interactive introduction to HTML and CSS
  • HTML Dog: HTML Tutorials — written guides to HTML for all levels

JavaScript

JavaScript is a programming language that makes websites interactive.

  • Codecademy: Introduction to JavaScript — interactive JavaScript guide for beginners
  • JavaScript Guide — comprehensive guide to JavaScript for intermediate programmers
  • JavaScript: The Good Parts — great book for experienced programmers

Ruby

Ruby is a clear and concise programming language

  • Codecademy: Introduction to Ruby — interactive course walking you through the Ruby language
  • Learn X in Y minutes: Ruby — Ruby syntax for experienced programmers

Git

  • Try Git — learn how to use the popular source control program

Paid Resources

  • TreeHouse ($9/month with student discount; 14-day free trial) — videos of HTML, CSS, iOS and Android development, and more
  • Code School ($19/month with student discount; many free courses) — interactive coding challenges, video lessons, and screencasts
  • General Assembly — on demand videos and courses from the makers of Dash

General

  • CS50 @ Harvard — introduction to computer science course from Harvard
  • Hacker News — similar to Reddit, for tech/startup news
  • WTF is a Hackathon — popular blog post explaining hackathons
  • JSFiddle — practice HTML/CSS/JavaScript in your browser
